# Know your shell

- Learn your default shell and config files
- Master navigation, piping, and redirection
- Create aliases, functions for repeat tasks
- Have a useful prompt
- Package managers
- Use fuzzy finders
- Multi pane with tmux
- Be lazy with git

<!-- end_slide -->
<!-- jump_to_middle -->

# Know your editor

- Pick one editor and go deep
- Vim keybindings
- Use linting, formatting, and snippets
- Master navigation, refactor, and search
- Configure keybindings to match your flow

<!-- end_slide -->
<!-- jump_to_middle -->

# Know your browser

- Learn DevTools panels and shortcuts
- Inspect network, storage, and console
- Debug performance with profiler
- Emulate devices and throttling
- Browser extensions for productivity
